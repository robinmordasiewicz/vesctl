Replace AWS VPC volterra site

Usage:
  vesctl site aws_vpc replace --name site-a [flags]

Aliases:
  replace, rep

Examples:
vesctl site aws_vpc replace

Flags:
      --allow-all-fwd-proxy          Enable forward proxy with this site and allow all request (only valid valid when --gw-type is ingress_egress_gw)
      --fwd-proxy-policies strings   List of forward proxy policies separated by comma, if give with arg "--allow-all-fwd-proxy" then this will be ignored
                                     		(only valid valid when --gw-type is ingress_egress_gw)
      --global-networks strings      List of global network connectors (only valid valid when --gw-type is ingress_egress_gw)
  -h, --help                         help for replace
      --inside-static-rt strings     List of simple static route for inside network (only valid valid when --gw-type is ingress_egress_gw)
  -n, --name string                  Name of the aws_vpc site
      --network-policies strings     List of network policies separated by comma (only valid valid when --gw-type is ingress_egress_gw)
      --outside-static-rt strings    List of simple static route for outside network (only valid valid when --gw-type is ingress_egress_gw)

Global Flags:
  -a, --cacert string         Server CA cert file path
  -c, --cert string           Client cert file path
      --config string         A configuration file to use for API gateway URL and credentials (default "/home/ubuntu/.vesconfig")
      --hardwareKey           Use yubikey for TLS connection
  -k, --key string            Client key file path
      --log-color             enable color for your logs (default true)
      --log-fabulous          enable fabulous writer for your logs (default true)
      --log-level int         Log Level for Site Deployment (default 3)
      --outfmt string         Output format for command
  -o, --output string         Output dir for command (default "./")
      --p12-bundle string     Client P12 bundle (key+cert) file path. Any password for this file should be in environment variable VES_P12_PASSWORD
  -u, --server-urls strings   API endpoint URL (default [http://localhost:8001])
      --show-curl             Emit requests from program in CURL format
      --timeout int           Timeout (in seconds) for command to finish (default 5)
